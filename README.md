## About

A growing collection of cool, elegant, efficient and performance-optimized
animation widgets.

## List of Animations

* Fade.

* HeartBeat.

* Rotate.

* Sonar.

### Usage Notes

Simply import `package:im_stepper/stepper.dart` and choose any animation.
